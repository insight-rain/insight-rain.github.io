## 摘要

Many embedded devices operate under resource constraints and in dynamic environments, requiring local decision-making capabilities. Enabling devices to make independent decisions in such environments can improve the responsiveness of the system and reduce the dependence on constant external control. In this work, we integrate an autonomous agent, programmed using AgentSpeak, with a small two-wheeled robot that explores a maze using its own decision-making and sensor data. Experimental results show that the agent successfully solved the maze in 59 seconds using 287 reasoning cycles, with decision phases taking less than one millisecond. These results indicate that the reasoning process is efficient enough for real-time execution on resource-constrained hardware. This integration demonstrates how high-level agent-based control can be applied to resource-constrained embedded systems for autonomous operation.

## 论文详细摘要

**论文标题**：在资源受限的机器人平台上嵌入自主智能体

### 1. 研究背景和动机
许多嵌入式设备在资源受限和动态环境中运行，需要本地的决策能力。传统的资源受限机器人平台（如基于微控制器的系统）通常采用集中式处理或简单的反应式代码，导致自主性有限、延迟较高。随着微控制器计算能力的提升，将**高级别的自主推理能力**（如基于智能体的编程）引入此类平台，有望显著提升系统的响应速度和独立性。本研究旨在探索将**信念-愿望-意图（BDI）智能体模型**嵌入到一个小型、低成本的轮式机器人中，以实现完全本地的自主决策。

### 2. 核心方法和技术创新
- **系统架构**：研究采用 **Embedded-BDI 框架**，该框架将用 **AgentSpeak 语言**编写的智能体逻辑，通过翻译引擎转换为优化的 C++ 代码，并与硬件相关的 C/C++ 代码（感知、动作函数）结合，生成可在微控制器上运行的二进制文件。
- **智能体设计**：在 Pololu 3pi+ 2040 机器人上实现了一个 BDI 智能体。智能体通过**反射传感器**感知迷宫环境（形成**信念**），以“到达终点”为**愿望**，并采用**左手定则**作为导航策略来生成和更新**意图**。
- **关键技术**：实现了**最小化的硬件控制 API**，使高级别的 AgentSpeak 脚本能方便地控制机器人底层运动，从而将面向智能体的编程范式与资源受限的嵌入式平台成功桥接。

### 3. 主要实验结果
- **任务性能**：智能体成功引导机器人在一个特制迷宫中沿最长路径行进，**总耗时 59 秒**，共执行了 **287 个推理周期**。
- **推理效率**：决策阶段（信念更新和计划选择）耗时极短，**平均少于 0.03 毫秒**，证明了推理过程的高效性。执行阶段（如电机控制）是周期耗时的主要部分（平均197毫秒），但这与具体任务相关。
- **资源占用**：最终编译的二进制文件对闪存和 RAM 的占用率分别仅为 **5.44%** 和 **6.25%**，表明该方案对硬件资源的需求极低。

### 4. 研究意义和价值
本研究成功验证了将 **BDI 自主智能体** 嵌入**资源高度受限的微控制器平台**的可行性。其价值在于：
- **实践验证**：证明了高级别、声明式的智能体编程范式可以满足嵌入式系统的实时性要求，为开发更智能、更自主的廉价机器人或物联网设备提供了新路径。
- **效率与轻量**：极短的决策时间和极低的资源占用表明，该框架非常适合在计算、内存和功耗均受限的边缘设备上实现实时自主决策。
- **未来应用**：为后续研究（如论文中提到的智能体间通信以实现协同探索）奠定了基础，展示了在资源受限平台上构建复杂多智能体系统的潜力。

### 实验目标
验证在资源受限的机器人平台上嵌入自主智能体（基于BDI模型和AgentSpeak语言）的可行性，并评估其在实时迷宫探索任务中的性能。

### 数据集与实验环境
- **数据集**：**未使用传统数据集**，而是使用**自定义的实体迷宫**（如图1所示）。
- **迷宫设计**：
  - 包含起点（S）、终点（E）和多个交叉点（如A、B）。
  - 设计为**最长路径**，以充分测试机器人的决策与导航能力。
  - 包含用于传感器校准的起始区域。
- **硬件平台**：Pololu 3pi+ 2040机器人（RP2040 MCU，264 kB SRAM，16 MB Flash）。

### 评价指标
1.  **任务完成时间**：机器人从起点到终点所需的总时间。
2.  **推理周期数**：智能体完成整个任务所执行的“感知-推理-行动”循环次数。
3.  **决策阶段耗时**：分解为三个核心阶段的执行时间：
    - **信念更新**：处理传感器数据、更新内部状态。
    - **计划选择**：根据信念和目标选择要执行的计划（行动序列）。
    - **意图执行**：执行具体动作（如移动、转向）。
4.  **内存占用**：编译后二进制文件对Flash和RAM的利用率。

### 基线方法对比
- **论文中未与传统的、非智能体的控制方法（如纯反应式C/C++代码）进行直接的定量性能对比**。
- **对比的焦点是概念性的**：论文将自身工作与两种现有方法进行对比，以突出其创新性：
    1.  **集中式处理/远程推理方案**：论文指出这类方案会导致**自主性有限和延迟较高**。本工作通过**本地嵌入智能体**解决了这些问题。

### 关键性能结果与结论
1.  **任务完成效果**：智能体**成功解开了迷宫**，总用时约 **59秒**，共经历了 **287个推理周期**。
2.  **决策效率**：
    - **信念更新**：平均 **0.004 ms**（最大0.022 ms）。
    - **计划选择**：平均 **0.024 ms**（最大0.153 ms）。
    - **决策总耗时（信念更新+计划选择）远低于1毫秒**，证明了推理过程的高效性。
    - **结论**：推理循环**足够高效，可用于资源受限硬件的实时决策**。
3.  **执行时间分析**：
    - **意图执行阶段**平均耗时 **197 ms**，远长于决策阶段。这是因为该阶段包含**传感器读取和电机控制等物理操作**，其时间**主要受迷宫物理结构（如直道长度）影响**，而非计算开销。
    - 这印证了系统的设计合理性：计算密集的推理部分极快，瓶颈在于物理执行，这符合嵌入式机器人系统的典型特征。

5.  **核心结论**：

    - 实验结果表明，**基于智能体的高级别控制可以应用于资源受限的嵌入式系统以实现自主操作**，为低成本、大规模自主机器人部署提供了新的软件范式。

### 未提供明确定量对比的原因
这是一篇**演示性论文**，其主要目的是**展示一种新方法（在MCU上运行BDI智能体）的可行性和潜力**，而非在特定任务上追求极致的性能优化或与成熟方案进行基准测试。其贡献更侧重于**系统集成、框架验证和概念证明**。未来的工作（如论文提到的多机器人协作）可能会涉及更复杂的性能比较。
